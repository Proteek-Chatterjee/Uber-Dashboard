# 🚖 Uber Dashboard – Power BI

An interactive **Power BI Dashboard** built using Uber ride booking data to analyze booking trends, revenue, customer behavior, vehicle performance, and location-based insights. The dashboard provides an intuitive interface for exploring key business metrics and supports data-driven decision-making.

---

## 📌 Project Overview

The dashboard is designed to help stakeholders monitor the overall performance of Uber operations by visualizing important KPIs and trends.

It consists of **four interactive pages**:

- 📊 Overview Dashboard
- 👥 Customer & Location Analysis
- 💰 Revenue Analysis
- 🚗 Vehicle Analysis

Each page contains dynamic visuals, slicers, and navigation buttons for seamless exploration.

---

## ✨ Features

- Interactive KPI Cards
- Dynamic Filtering
- Month & Quarter Toggle
- Vehicle Type Navigation
- Customer Analysis
- Revenue Insights
- Pickup & Drop Location Analysis
- Performance Comparison Across Vehicle Types
- Responsive Dashboard Layout
- Minimal Uber-inspired UI Design

# 📊 Dashboard Pages

## 1️⃣ Overview Dashboard

Provides a quick summary of the entire business.

### KPIs

- Total Bookings
- Completed Bookings
- Lost Bookings
- Total Revenue
- Total Distance
- Average Distance
- Customer Rating
- Driver Rating

### Visualizations

- Completed Bookings by Quarter
- Revenue by Month
- Revenue by Vehicle Type
- KPI Summary Cards

---

## 2️⃣ Customer & Location Dashboard

Focuses on customer activity and geographical booking distribution.

### Visualizations

- Customer Count by Month
- Payment Method Distribution
- Booking by Pickup Location
- Booking by Drop Location

Provides insights into:

- Customer growth
- Popular pickup locations
- Popular drop locations
- Preferred payment methods

---

## 3️⃣ Revenue Dashboard

Analyzes revenue generation across different dimensions.

### Visualizations

- Revenue by Month
- Revenue by Vehicle Type
- Revenue by Payment Method
- Revenue by Customer

Helps answer:

- Which vehicle earns the most revenue?
- Which payment method contributes the highest revenue?
- Monthly revenue trends
- Highest revenue-generating customers

---

## 4️⃣ Vehicle Dashboard

Compares the performance of all Uber vehicle categories.

### Vehicle Types

- Auto
- Bike
- Go Mini
- Go Sedan
- Premier Sedan
- Uber XL

### Visualizations

- Revenue Trend by Vehicle Type
- Vehicle-wise Performance Table

Metrics include:

- Customer Count
- Completed Bookings
- Lost Bookings
- Revenue
- Average Distance

---

# 📈 Key Metrics

| Metric | Description |
|----------|-------------|
| Total Bookings | Total rides booked |
| Completed Bookings | Successfully completed rides |
| Lost Bookings | Cancelled/Failed rides |
| Revenue | Total earnings |
| Total Distance | Distance travelled by all rides |
| Average Distance | Average ride distance |
| Customer Rating | Average customer satisfaction |
| Driver Rating | Average driver performance |
